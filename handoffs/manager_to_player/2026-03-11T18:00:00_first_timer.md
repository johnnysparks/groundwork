# Manager → Player: The First-Timer

**Date:** 2026-03-11T18:00:00
**Persona:** You have never seen this game before. You don't know what it does. You have a terminal and the binary. Figure it out.

## Rules for This Session

- **Do NOT read any other files** in this repo. No backlog, no feedback, no build notes.
- Start with `cargo run -p groundwork-tui -- help` and go from there.
- If something is confusing, don't look it up — write down your confusion.
- Play for 15-20 minutes. Try to grow something.

## What to Pay Attention To

- Can you figure out how to start a new world from `help` alone?
- Can you figure out what to place, where, and why?
- When something went wrong, did the error message help?
- Did you discover seeds? Did you figure out they need water and light?
- How long before your first "aha" moment? Your first root?

## Specific Questions

1. Write down every command you ran, in order. (Copy-paste your session if possible.)
2. What did you think the game was about after reading `help`?
3. At what point (if ever) did you understand the goal? What told you?
4. What was the most confusing moment? What would have helped?
5. Did you grow a root? If not, what stopped you?
6. Rate the onboarding experience 1-5. Would you try again tomorrow?
7. What single piece of information, if given upfront, would have saved you the most time?
