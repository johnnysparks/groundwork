## Summary

Fixed the P0 bug where all leaf voxels had water_level=0, making foliage health tinting non-functional. Root cause: stale WASM binary in `public/wasm/` predating the health-writing code. The `npm run wasm` script now auto-syncs to `public/wasm/`.

## Key Finding

The Rust sim correctly writes tree health to leaf voxel water_level (proven by unit test). The bug was purely a deployment issue — the Vite dev server served a months-old WASM binary because `public/` directory takes precedence over source `wasm/` directory.
