## Architecture Notes

- Voxel byte 3 (nutrient_level) = species_id on Trunk/Leaf/Branch/Root
- Voxel byte 1 (water_level) = health (0-255) on Leaf/Branch (re-rasterized every 50 ticks)
- Gnome is JS-side only (TaskQueue + GardenerSprite). Executive mandate wants sim-side Rust entity.
- Save/load = base64 of raw voxel grid via WASM linear memory read/write
- Pioneer succession = every 50 ticks, 20 random samples, soil → moss → grass → wildflower
